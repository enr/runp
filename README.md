For examples see [examples directory](examples/) or the [documentation](https://enr.github.io/runp/).


## Develop

Download or clone repository.

Build (binaries will be created in `bin/`):

```
./.sdlc/build
```

Check (code quality and tests):

```
./.sdlc/check
```
